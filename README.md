# Mon Vieux Grimoire – Back-end

Ce projet correspond au développement du back-end d’une application de notation de livres, réalisée dans le cadre de la formation OpenClassrooms.

L’API permet de gérer des utilisateurs, des livres et des notes.

## Fonctionnalités

- Inscription et connexion des utilisateurs
- Création, modification et suppression de livres
- Notation des livres (une seule note par utilisateur)
- Upload et optimisation des images

## Technologies utilisées

- Node.js / Express
- MongoDB / Mongoose
- JWT / bcrypt
- Multer / Sharp
